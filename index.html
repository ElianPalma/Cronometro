Operadores logicos

!= Sirve para ver si dos variables son diferentes

&& indica que todos los valores (habitualmente dos comparaciones) deben de ser true para que ejecute

|| Indica si cualquier de los valores a comparar es true, este ejecuta

! Modifica si es true a false y si es false a true
